## 核心模块

### 1. 网络搜索

```
Bocha AI Search (主) → Bing 中国版 (回退)
       │
       ├─ 流式输出（stream: true，3s首token）
       ├─ 结构化数据（天气/股票/赛事自动识别）
       └─ AI 回答（直接推给用户，不经过DeepSeek二次总结）
```

### 2. 意图分类

```
DeepSeek 直接分类（主） → 快速规则拦截 → k-NN 回退
```

### 3. RAG检索

```
ChromaDB(稠密) + BM25(jieba分词) → RRF融合(k=60) → Top-5
```

### 4. 用户认证

```
注册: bcrypt哈希 → SQLite users表
登录: 验证 → JWT(24h) → 所有端点 Bearer Token
会话: session_registry.user_id 隔离
```

### 5. 性能优化

```
并行: asyncio.create_task（意图+搜索同时进行）
流式: Bocha stream → 逐字推送
缓存: QueryCache LRU 100条
```
