# AI Engineer Mentor（课程助手）— MVP 思维导图
- 产品定位
  - 名称：AI Engineer Mentor
  - 目标用户：AIGC 大模型应用工程师学习者
  - 核心价值：围绕课程资料提供智能问答辅助学习
  - 面试展示项目
- 核心目标
  - 智能 RAG 课程问答：基于课程资料精准检索+生成
  - 文件上传即时解析：支持6种格式自动分块入库
  - 意图识别智能路由：区分课程问题与闲聊，减少无效检索
  - 多会话上下文管理：按日期分组，会话历史持久化
  - 流式输出体验：SSE 逐 token 返回，打字机效果
- MVP 功能模块
  - RAG 知识库问答
    - ChromaDB 向量检索（Top-3）
    - instructor-xl 本地嵌入（零外部依赖）
    - DeepSeek 云端 API 流式生成
    - LangChain 编排链路
    - 余弦相似度阈值过滤无关检索
  - 文件上传解析
    - PDF: PyMuPDF（fitz）
    - DOCX: python-docx
    - TXT: 直接读取
    - HTML: BeautifulSoup 去标签
    - PPTX: python-pptx 提取文本
    - IPYNB: JSON 解析 code/markdown cells
    - 自动分块入库 ChromaDB
  - 意图识别路由
    - 方案：嵌入向量余弦相似度 + 会话级缓存
    - 课程锚点 vs 闲聊锚点打分
    - RAG_THRESHOLD=0.69 精准过滤
    - 同会话仅首次计算嵌入（缓存复用）
  - 对话与多会话管理
    - 新建/切换/删除会话
    - 按日期分组（今天/昨天/前7天/更早）
    - SQLite 持久化历史（最近10轮）
    - session_id UUID 前后端同步
  - UI 交互层
    - Gradio Chatbot 流式显示
    - 玻璃拟态顶栏 + 渐变色侧边栏
    - 退出登录按钮
    - 文件拖拽上传
    - 路由标记显示（📚参考 / 💬闲聊）
- 技术栈
  - 前端：Gradio 6.18 + httpx SSE 流式
  - 后端：FastAPI + LangChain + Pydantic
  - LLM：DeepSeek 云端 API（deepseek-chat）
  - 嵌入：instructor-xl（sentence-transformers 本地）
  - 向量库：ChromaDB（PersistentClient + cosine 空间）
  - 文件解析：PyMuPDF + python-docx + python-pptx + BeautifulSoup
  - 文本分块：RecursiveCharacterTextSplitter（chunk=500, overlap=50）
  - 业务存储：SQLite（标准库）
  - 配置：python-dotenv .env
- 工程实践
  - 虚拟环境隔离（pbl_venv）
  - 增量迭代开发（不修改已有代码文件）
  - 知识库可重建（删 chroma_db/ 目录即重置）
  - 意图路由会话级缓存
  - 前后端分离：Gradio ↔ FastAPI SSE
  - GitHub 版本控制
- 下一步
  - 上传真实课程资料构建知识库
  - 压测 RAG 链路（并发、延迟）
  - 简历项目描述 + 面试问答准备
  - A4 项目实战报告撰写
