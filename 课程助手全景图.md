## 一、产品定位

| 维度 | 描述 |
|------|------|

## 二、MVP 功能全景（优先级 P0 → P3）

```
课程助手 MVP
├── P0 🔴 RAG 知识库问答
│   ├── 课程资料解析（PDF / Word）
│   ├── 文本分块（RecursiveCharacterTextSplitter，500/50）
│   ├── 向量检索（instructor-xl + ChromaDB，Top-3）
│   ├── LLM 生成（DeepSeek-Chat 云端 API，流式 SSE）
│   └── 引用来源标注（📚 参考：xxx）
│
├── P1 🟠 文件上传解析
│   ├── PDF（PyMuPDF）+ Word（python-docx）
│   ├── 自动提取文本 → 分块 → 入库 ChromaDB
│   └── 文件大小限制 50MB
│
├── P2 🟡 意图识别路由
│   ├── 嵌入向量余弦相似度（instructor-xl 本地计算）
│   ├── RAG 锚点（11 个课程主题）+ 闲聊锚点（9 个场景）
│   ├── 自动分流：课程问题 → RAG / 闲聊 → 直接 LLM
│   └── 会话级缓存（同会话仅计算一次，≈0s 延迟）
│
└── P3 🟢 基础对话
    ├── 流式 SSE 响应（逐字输出）
    ├── 多会话管理（UUID + 日期分组切换）
    ├── 对话历史记忆（最近 10 轮，SQLite 持久化）
    ├── 用户名识别与记忆
    └── 课程导师人设（System Prompt）
```

## 四、核心数据流

```
用户输入（文本 / 文件）
    │
    ▼
意图路由（余弦相似度，缓存优化）
    │
    ├── 课程问答 ──→ ChromaDB 检索 Top-3 ──→ 上下文拼接 ──→ DeepSeek 生成
    │                                                         │
    ├── 日常闲聊 ──→ 对话历史 + System Prompt ──→ DeepSeek 直接生成
    │                                                         │
    └── 文件上传 ──→ PyMuPDF/docx 解析 ──→ 分块 ──→ 入库 ChromaDB
                                                              │
                                                              ▼
                                                     SSE 流式返回
                                                   （含 📚 引用来源）
```
